# Code of Conduct

The veeox project follows the **Rust Code of Conduct**.

This describes the minimum behavior expected from all contributors and community members.

## Enforcement

Instances of violations can be reported by contacting the project maintainer(s). If you need a
private channel, use the contact method listed on the maintainer's GitHub profile (or request a
private follow-up in a GitHub Discussion).

We will investigate and take appropriate action.

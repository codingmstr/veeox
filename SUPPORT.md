# Support

Need help? Here is the fastest path to a solution.

## Where to go?

| I need to...                      | Go to...                           |
| :-------------------------------- | :--------------------------------- |
| Ask “How do I…?” or discuss ideas | Discussions (`../../discussions`)  |
| Report a reproducible bug         | Issues (`../../issues`)            |
| Report a security vulnerability   | Security Policy (`../SECURITY.md`) |

---

## Expectations

we are community-driven.

- **Bugs:** regressions and blockers get priority.
- **Features:** discuss first in Discussions.
- **Response:** high-quality reports (with repro) get faster responses.
